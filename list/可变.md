列表的“可变性”是其另一核心特性，具体体现为两方面：

1. **元素可修改**  
   列表创建后，可通过下标直接修改任意位置的元素，修改操作会直接作用于原列表（而非生成新列表）。  
   示例：  
   ```python
   nums = [1, 2, 3]
   nums[1] = 20  # 直接修改下标1的元素
   print(nums)  # 输出：[1, 20, 3]（原列表被改变）
   ```

2. **长度可动态变化**  
   列表的长度会随元素的添加（如 `append()`、`insert()`）或删除（如 `remove()`、`pop()`）自动调整，无需预先定义固定大小。  
   示例：  
   ```python
   arr = []  # 初始长度为0
   arr.append(1)  # 添加元素，长度变为1
   arr.append(2)  # 长度变为2
   arr.pop()      # 删除元素，长度变回1
   print(len(arr))  # 输出：1
   ```

这两方面共同构成了列表的可变性，使其能够灵活应对数据的动态变化，与元组（元素和长度均不可变）形成鲜明对比。
