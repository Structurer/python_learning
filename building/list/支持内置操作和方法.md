列表支持丰富的内置操作和方法，这些功能基于其有序性、可变性等特性设计，可分为**内置函数**、**运算符**和**列表对象方法**三类，具体如下：


### 一、内置函数（可直接作用于列表的全局函数）
1. `len(list)`：返回列表的长度（元素个数）  
   ```python
   print(len([1, 2, 3]))  # 输出：3
   ```

2. `max(list)` / `min(list)`：返回列表中的最大/最小值（要求元素类型可比较，如数字、同类型字符串）  
   ```python
   nums = [5, 2, 8]
   print(max(nums))  # 输出：8
   print(min(nums))  # 输出：2
   ```

3. `sum(list)`：返回列表中所有数字元素的和（仅适用于全数字列表）  
   ```python
   print(sum([1, 2, 3]))  # 输出：6
   ```

4. `sorted(list)`：返回一个排序后的新列表（原列表不变，与列表自带的 `sort()` 方法不同）  
   ```python
   nums = [3, 1, 2]
   print(sorted(nums))  # 输出：[1, 2, 3]（新列表）
   print(nums)          # 原列表不变：[3, 1, 2]
   ```


### 二、运算符（可直接对列表进行的运算）
1. `+`：拼接两个列表，返回新列表（原列表不变）  
   ```python
   a = [1, 2]
   b = [3, 4]
   print(a + b)  # 输出：[1, 2, 3, 4]
   ```

2. `*`：重复列表元素指定次数，返回新列表  
   ```python
   print([1, 2] * 3)  # 输出：[1, 2, 1, 2, 1, 2]
   ```

3. `in` / `not in`：判断元素是否在列表中，返回布尔值  
   ```python
   fruits = ["apple", "banana"]
   print("apple" in fruits)     # 输出：True
   print("orange" not in fruits)  # 输出：True
   ```

4. 索引运算符 `[]`：通过下标访问单个元素（支持正索引、负索引）  
   ```python
   nums = [10, 20, 30]
   print(nums[1])   # 输出：20（正索引）
   print(nums[-1])  # 输出：30（负索引）
   ```

5. 切片运算符 `[start:end:step]`：获取子列表（返回新列表）  
   ```python
   nums = [0, 1, 2, 3, 4]
   print(nums[1:4])  # 输出：[1, 2, 3]
   ```


### 三、列表对象的内置方法（仅列表可调用的方法）
1. **增删元素相关**  
   - `append(x)`：在列表末尾添加元素 `x`（原地修改）  
     ```python
     fruits = ["apple"]
     fruits.append("banana")  # 变为：["apple", "banana"]
     ```
   - `insert(i, x)`：在索引 `i` 处插入元素 `x`（后续元素后移，原地修改）  
     ```python
     fruits.insert(1, "orange")  # 变为：["apple", "orange", "banana"]
     ```
   - `remove(x)`：删除第一个出现的元素 `x`（若不存在则报错，原地修改）  
     ```python
     fruits.remove("orange")  # 变为：["apple", "banana"]
     ```
   - `pop(i)`：删除并返回索引 `i` 处的元素（默认删除最后一个，原地修改）  
     ```python
     last = fruits.pop()  # last 为 "banana"，列表变为：["apple"]
     ```
   - `clear()`：清空列表（原地修改，变为空列表）  
     ```python
     fruits.clear()  # 变为：[]
     ```

2. **排序与反转**  
   - `sort(key=None, reverse=False)`：对列表排序（原地修改，默认升序）  
     ```python
     nums = [3, 1, 2]
     nums.sort()  # 变为：[1, 2, 3]
     nums.sort(reverse=True)  # 变为：[3, 2, 1]（降序）
     ```
   - `reverse()`：反转列表元素顺序（原地修改）  
     ```python
     nums = [1, 2, 3]
     nums.reverse()  # 变为：[3, 2, 1]
     ```

3. **其他常用方法**  
   - `count(x)`：返回元素 `x` 在列表中出现的次数  
     ```python
     print([1, 2, 2, 3].count(2))  # 输出：2
     ```
   - `index(x)`：返回元素 `x` 第一次出现的索引（若不存在则报错）  
     ```python
     print(["a", "b", "a"].index("a"))  # 输出：0
     ```
   - `copy()`：返回列表的浅拷贝（新列表，修改新列表不影响原列表）  
     ```python
     original = [1, 2]
     new_list = original.copy()
     new_list.append(3)
     print(original)  # 原列表不变：[1, 2]
     ```


这些内置操作和方法极大地简化了对列表的处理，涵盖了元素的增删改查、排序、拼接等常见需求，是列表灵活性的重要体现。
