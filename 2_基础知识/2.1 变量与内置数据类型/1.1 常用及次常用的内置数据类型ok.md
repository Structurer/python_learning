# Python 内置类型转换函数 - 输入输出对照表
| 函数                | 支持的输入数据类型                                                                 | 输出结果格式/类型                          | 核心示例（输入→输出）                          |
|---------------------|----------------------------------------------------------------------------------|-------------------------------------------|-----------------------------------------------|
| `int(x[, base])`    | 1. 浮点数<br>2. 合法数字字符串（可指定base进制）<br>3. 布尔值<br>4. 可转整数的对象 | 十进制整数                                | int(3.9)→3<br>int("10",2)→2<br>int(True)→1     |
| `float(x)`          | 1. 整数<br>2. 合法浮点数字符串（含科学计数法）<br>3. 布尔值<br>4. 可转浮点数的对象 | 浮点数（带小数点）                        | float(5)→5.0<br>float("3.14e2")→314.0<br>float(False)→0.0 |
| `complex(real[, imag])` | 1. real：整数/浮点数/布尔值<br>2. imag（可选）：整数/浮点数/布尔值（默认0）        | 复数（a+bj格式）                          | complex(2,3.5)→2+3.5j<br>complex(True)→1+0j   |
| `str(x)`            | 任意Python对象（数值、容器、自定义对象等）                                        | 面向用户的可读字符串                      | str([1,2,3])→"[1, 2, 3]"<br>str({"name":"Tom"})→"{'name': 'Tom'}" |
| `repr(x)`           | 任意Python对象                                                                    | 面向解释器的表达式字符串（可直接执行）    | repr("hello")→"'hello'"<br>repr((1,2))→"(1, 2)" |
| `eval(str)`         | 合法的Python表达式字符串（运算式、容器定义等）                                    | 表达式执行后的结果对象（类型随表达式）    | eval("1+2*3")→7<br>eval("[x*2 for x in range(3)]")→[0,2,4] |
| `tuple(s)`          | 可迭代对象（列表、字符串、集合、生成器、字典等）                                  | 不可变元组                                | tuple([1,2,3])→(1,2,3)<br>tuple("abc")→('a','b','c') |
| `list(s)`           | 可迭代对象（元组、字符串、集合、生成器、字典等）                                  | 可变列表                                  | list((1,2,3))→[1,2,3]<br>list({1,2,3})→[1,2,3]（顺序不固定） |
| `set(s)`            | 可迭代对象（列表、字符串、元组、生成器等）                                        | 可变集合（自动去重，无固定顺序）          | set([1,2,2,3])→{1,2,3}<br>set("aabcc")→{'a','b','c'} |
| `dict(d)`           | (key, value)形式的可迭代对象（元组序列、列表序列等）<br>关键字参数（如dict(name="Tom")） | 字典（key唯一，无序）                     | dict([("name","Tom"), ("age",18)])→{'name':'Tom', 'age':18} |
| `frozenset(s)`      | 可迭代对象（列表、字符串、元组、生成器等）                                        | 不可变集合（无法增删元素）                | frozenset([1,2,3])→frozenset({1,2,3})<br>frozenset("abc")→frozenset({'a','b','c'}) |
| `chr(x)`            | 非负整数（Unicode编码，范围0~1114111）                                            | 单个字符（长度为1的字符串）               | chr(65)→'A'<br>chr(20013)→'中'                 |
| `ord(x)`            | 单个字符（长度为1的字符串）                                                       | 字符对应的Unicode整数值                   | ord('A')→65<br>ord('中')→20013                 |
| `hex(x)`            | 整数（正数/负数）                                                                 | 带0x前缀的十六进制字符串                  | hex(10)→'0xa'<br>hex(-255)→'-0xff'             |
| `oct(x)`            | 整数（正数/负数）                                                                 | 带0o前缀的八进制字符串                    | oct(8)→'0o10'<br>oct(-63)→'-0o77'              |

### 关键点回顾
1. 输入类型核心规律：除`chr/ord/hex/oct`有严格类型限制外，多数转换函数支持**可迭代对象**或**基础数值/布尔类型**；
2. 输出类型关键特征：`int/float/complex`输出数值类型，`str/repr`输出字符串，`tuple/list/set/dict/frozenset`输出容器类型，`chr/ord/hex/oct`输出字符/字符串/整数；
3. 特殊注意：`eval`需谨慎使用（避免执行恶意字符串），`set/frozenset`会自动去重且无固定顺序。
