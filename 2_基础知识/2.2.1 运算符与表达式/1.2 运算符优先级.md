### 优先级
运算符的执行顺序，（从高到低排列，同一行优先级相同，运算时从左到右结合，除非有特殊说明）：

1. **括号** `()`  
   - 功能：强制改变运算顺序，括号内的表达式优先计算，支持多层嵌套（如 `(a + (b * c)) / d`）。  
   - 示例：`(2 + 3) * 4` 先算 `2 + 3 = 5`，再算 `5 * 4 = 20`；若去掉括号 `2 + 3 * 4` 则先算乘法，结果为 `14`。  


2. **索引/切片/属性访问**  
   - `x[index]`（索引）：访问序列（列表、元组、字符串等）中指定位置的元素，`index` 为索引值（从0开始）。  
     示例：`lst = [10, 20, 30]; lst[1]` 结果为 `20`。  
   - `x[index:index]`（切片）：截取序列中从起始索引到结束索引（不含）的子序列，支持省略索引（如 `x[:3]` 取前3个元素，`x[2:]` 取从第3个元素开始的所有元素）。  
     示例：`s = "hello"; s[1:4]` 结果为 `"ell"`。  
   - `x.attribute`（属性访问）：访问对象的属性（如类实例的变量或方法）。  
     示例：`str.lower("HELLO")` 中 `.lower` 是字符串对象的方法属性。  
   - `x(args...)`（函数调用）、`f(x)`（函数应用）：调用函数或方法，`args...` 为传入的参数。  
     示例：`print("hi")` 调用 `print` 函数；`lst.append(40)` 调用列表的 `append` 方法。  


3. **幂运算** `**`  
   - 功能：计算一个数的几次方，右结合（即多个幂运算从右往左计算）。  
   - 示例：`2** 3 **2` 等价于 `2** (3 **2) = 2** 9 = 512`；`4 **0.5` 表示4的平方根，结果为 `2.0`。  


4. **按位取反** `~x`  
   - 功能：对整数 `x` 的二进制位进行取反（0变1，1变0），运算规则为 `~x = -(x + 1)`。  
   - 示例：`x = 3`（二进制 `000...0011`），`~x` 结果为 `-4`（二进制 `111...1100`）。  


5. **正负号** `+x`（正）、`-x`（负）  
   - 功能：`+x` 表示 `x` 本身（通常可省略）；`-x` 表示 `x` 的相反数。  
   - 示例：`+5` 结果为 `5`；`-3` 结果为 `-3`；`a = 10; -a` 结果为 `-10`。  


6. **乘法、除法、取模、整除**  
   - `*`（乘法）：计算两个数的乘积，或用于序列重复（如 `[1] * 3` 结果为 `[1, 1, 1]`）。  
     示例：`3 * 4 = 12`；`"ab" * 2 = "abab"`。  
   - `/`（除法）：计算两个数的商，结果为浮点数。  
     示例：`7 / 2 = 3.5`；`10 / 5 = 2.0`。  
   - `%`（取模）：计算两个数相除的余数，结果的符号与除数一致。  
     示例：`7 % 3 = 1`；`-7 % 3 = 2`（因除数为正，余数为正）。  
   - `//`（整除）：计算两个数相除的商并向下取整（向负无穷方向取整）。  
     示例：`7 // 3 = 2`；`-7 // 3 = -3`（向下取整为 `-3`）。  


7. **加法、减法** `+`、`-`  
   - `+`：计算两个数的和，或用于序列拼接（如 `[1,2] + [3] = [1,2,3]`）、字符串拼接（如 `"a" + "b" = "ab"`）。  
     示例：`5 + 3 = 8`；`"hello" + " world" = "hello world"`。  
   - `-`：计算两个数的差，或用于集合差集（如 `{1,2,3} - {2} = {1,3}`）。  
     示例：`10 - 4 = 6`；`a = 5; b = 2; a - b = 3`。  


8. **按位左移、右移** `<<`、`>>`  
   - `<<`（左移）：将整数的二进制位向左移动指定位数，右边补0，等价于 `x * (2 **n)`（`n` 为移动位数）。  
     示例：`3 << 2`（3的二进制 `11` 左移2位为 `1100`），结果为 `12`（即 `3 * 2** 2 = 12`）。  
   - `>>`（右移）：将整数的二进制位向右移动指定位数，左边补符号位（正数补0，负数补1），等价于 `x // (2 **n)`（`n` 为移动位数）。  
     示例：`8 >> 1`（8的二进制 `1000` 右移1位为 `100`），结果为 `4`（即 `8 // 2** 1 = 4`）。  


9. **按位与** `&`  
   - 功能：对两个整数的二进制位进行“与”操作，对应位均为1则结果为1，否则为0。  
   - 示例：`3 & 5`（3是 `011`，5是 `101`），结果为 `001`（即 `1`）。  


10. **按位异或** `^`  
    - 功能：对两个整数的二进制位进行“异或”操作，对应位不同则结果为1，相同则为0。  
    - 示例：`3 ^ 5`（3是 `011`，5是 `101`），结果为 `110`（即 `6`）。  


11. **按位或** `|`  
    - 功能：对两个整数的二进制位进行“或”操作，对应位至少有一个为1则结果为1，否则为0。  
    - 示例：`3 | 5`（3是 `011`，5是 `101`），结果为 `111`（即 `7`）。  


12. **判断运算符**  
    - `<`、`<=`、`>`、`>=`、`!=`、`==`（比较判断）：比较两个对象的值或大小关系，返回布尔值（`True`/`False`）。  
      示例：`5 > 3` 结果为 `True`；`"abc" == "abd"` 结果为 `False`。  
    - `is`、`is not`（身份判断）：判断两个变量是否指向同一个对象（即内存地址是否相同），返回布尔值。  
      示例：`a = [1]; b = a; a is b` 结果为 `True`（指向同一列表）；`c = [1]; a is c` 结果为 `False`（不同对象）。  
    - `in`、`not in`（成员判断）：判断一个元素是否属于某个容器（列表、字符串、集合等），返回布尔值。  
      示例：`2 in [1,2,3]` 结果为 `True`；`"x" not in "abc"` 结果为 `True`。  


13. **逻辑非** `not x`  
    - 功能：对布尔值 `x` 取反，`not True` 为 `False`，`not False` 为 `True`；非布尔值会先转换为布尔值再取反（如 `0`、`""`、`None` 视为 `False`，其余视为 `True`）。  
    - 示例：`not 5 > 3` 结果为 `False`；`not []` 结果为 `True`（空列表视为 `False`，取反为 `True`）。  


14. **逻辑与** `and`  
    - 功能：对两个条件进行“与”判断，左右两边均为 `True` 时结果为 `True`，否则为 `False`；左结合（从左到右计算），且具有短路特性（左边为 `False` 时，右边不再计算）。  
    - 示例：`3 > 1 and 5 < 10` 结果为 `True`；`3 > 5 and 5 < 10` 结果为 `False`（左边为 `False`，右边不计算）。  


15. **逻辑或** `or`  
    - 功能：对两个条件进行“或”判断，左右两边至少有一个为 `True` 时结果为 `True`，否则为 `False`；左结合，且具有短路特性（左边为 `True` 时，右边不再计算）。  
    - 示例：`3 > 5 or 5 < 10` 结果为 `True`（右边为 `True`）；`3 > 1 or 5 < 10` 结果为 `True`（左边为 `True`，右边不计算）。  


16. **赋值运算符**  
    - 包含：`=`、`+=`、`-=`、`*=`、`/=`、`//=`、`%=`、`**=`、`&=`、`|=`、`^=`、`<<=`、`>>=`。  
    - 功能：将右边的值赋给左边的变量，复合赋值运算符（如 `+=`）等价于“左边变量 = 左边变量 运算符 右边值”。  
    - 结合性：右结合（如 `a = b = 5` 等价于 `b = 5; a = b`）。  
    - 示例：`x = 10`（基本赋值）；`x += 5` 等价于 `x = x + 5`（结果 `x = 15`）；`y *= 3` 等价于 `y = y * 3`。  


17. **lambda 表达式** `lambda`  
    - 功能：创建匿名函数，语法为 `lambda 参数: 表达式`，返回函数对象，优先级最低（最后执行）。  
    - 示例：`add = lambda a, b: a + b` 定义一个匿名函数，`add(2, 3)` 结果为 `5`；在复杂表达式中，`lambda` 会在其他运算完成后执行。
