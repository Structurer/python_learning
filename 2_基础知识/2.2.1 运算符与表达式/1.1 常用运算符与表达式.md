Python 中的运算符用于执行各种操作（如算术计算、比较、逻辑判断等），按功能可分为以下几类：


### 一、算术运算符
用于基本数学计算，操作数为数值。
| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `3 + 5` | `8` |
| `-` | 减法 | `10 - 4` | `6` |
| `*` | 乘法 | `2 * 6` | `12` |
| `/` | 除法（结果为浮点数） | `10 / 3` | `3.333...` |
| `//` | 整除（向下取整） | `10 // 3` | `3`（整数）；`-10 // 3` → `-4` |
| `%` | 取模（求余数） | `10 % 3` | `1`；`7 % 2` → `1` |
| `**` | 幂运算（次方） | `2** 3` | `8`（2的3次方）；`9 **0.5` → `3.0`（开平方） |


### 二、赋值运算符
用于给变量赋值，可结合算术运算符简化操作（复合赋值）。
| 运算符 | 描述 | 示例 | 等价于 |
|--------|------|------|--------|
| `=` | 基础赋值 | `a = 5` | - |
| `+=` | 加法赋值 | `a += 3` | `a = a + 3` |
| `-=` | 减法赋值 | `a -= 2` | `a = a - 2` |
| `*=` | 乘法赋值 | `a *= 4` | `a = a * 4` |
| `/=` | 除法赋值 | `a /= 2` | `a = a / 2` |
| `//=` | 整除赋值 | `a //= 3` | `a = a // 3` |
| `%=` | 取模赋值 | `a %= 5` | `a = a % 5` |
| `**=` | 幂运算赋值 | `a**= 2` | `a = a ** 2` |


### 三、比较运算符
用于比较两个值的关系，返回布尔值（`True` 或 `False`）。
| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | `True`；`"a" == "b"` → `False` |
| `!=` | 不等于 | `3 != 5` | `True` |
| `>` | 大于 | `10 > 7` | `True` |
| `<` | 小于 | `4 < 2` | `False` |
| `>=` | 大于等于 | `5 >= 5` | `True` |
| `<=` | 小于等于 | `3 <= 1` | `False` |


### 四、逻辑运算符
用于组合多个条件判断，操作数为布尔值或可转换为布尔值的表达式，返回布尔值。
| 运算符 | 描述 | 规则 | 示例 | 结果 |
|--------|------|------|------|------|
| `and` | 逻辑与 | 两边都为 `True` 才返回 `True` | `(3 > 2) and (5 < 10)` | `True` |
| `or` | 逻辑或 | 至少一边为 `True` 就返回 `True` | `(3 > 5) or (5 < 10)` | `True` |
| `not` | 逻辑非 | 取反（`True` 变 `False`，反之亦然） | `not (3 > 5)` | `True` |


### 五、成员运算符
判断元素是否在序列（列表、字符串、元组等）中，返回布尔值。
| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `in` | 元素在序列中 | `3 in [1, 2, 3]` | `True`；`"a" in "abc"` → `True` |
| `not in` | 元素不在序列中 | `5 not in [1, 2, 3]` | `True` |


### 六、身份运算符
判断两个变量是否指向**同一个对象**（内存地址相同），而非值是否相等。
| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `is` | 是同一个对象 | `a = [1,2]；b = a；a is b` | `True`（`a`和`b`指向同一列表） |
| `is not` | 不是同一个对象 | `a = [1,2]；b = [1,2]；a is not b` | `True`（值相等但为不同对象） |

> 注意：`==` 比较值是否相等，`is` 比较内存地址是否相同（即是否为同一对象）。


### 七、位运算符（了解即可）
对整数的二进制位进行操作，较少用于日常开发。
| 运算符 | 描述 | 示例（二进制） | 结果（十进制） |
|--------|------|---------------|----------------|
| `&` | 按位与（对应位都为1则为1） | `5 (101) & 3 (011)` → `001` | `1` |
| `|` | 按位或（对应位有1则为1） | `5 (101) | 3 (011)` → `111` | `7` |
| `^` | 按位异或（对应位不同则为1） | `5 (101) ^ 3 (011)` → `110` | `6` |
| `~` | 按位取反（0变1，1变0） | `~5 (000...0101)` → `111...1010` | `-6` |
| `<<` | 左移（二进制位左移n位，右边补0） | `5 (101) << 1` → `1010` | `10` |
| `>>` | 右移（二进制位右移n位，左边补符号位） | `5 (101) >> 1` → `10` | `2` |


### 优先级
运算符的执行顺序，（从高到低排列，同一行优先级相同，运算时从左到右结合，除非有特殊说明）：

1. **括号** `()`（最高优先级，用于强制改变运算顺序）  
2. **索引/切片/属性访问**  
   - `x[index]`（索引）、`x[index:index]`（切片）  
   - `x.attribute`（属性访问）  
   - `x(args...)`（函数调用）、`f(x)`（函数应用）  
3. **幂运算** `**`（右结合，如 `2** 3 **2` 等价于 `2** (3 **2)`）  
4. **按位取反** `~x`  
5. **正负号** `+x`（正）、`-x`（负）  
6. **乘法、除法、取模、整除**  
   - `*`（乘法）、`/`（除法）、`%`（取模）、`//`（整除）  
7. **加法、减法** `+`、`-`  
8. **按位左移、右移** `<<`、`>>`  
9. **按位与** `&`  
10. **按位异或** `^`  
11. **按位或** `|`  
12. **判断运算符**  
    - `<`、`<=`、`>`、`>=`、`!=`、`==`  （比较判断）
    - `is`、`is not`（身份判断）  
    - `in`、`not in`（成员判断）  
13. **逻辑非** `not x`  
14. **逻辑与** `and`（左结合，如 `a and b and c` 等价于 `(a and b) and c`）  
15. **逻辑或** `or`（左结合，如 `a or b or c` 等价于 `(a or b) or c`）  

16. **赋值运算符**  
    - 包含：`=`、`+=`、`-=`、`*=`、`/=`、`//=`、`%=`、`**=`、`&=`、`|=`、`^=`、`<<=`、`>>=`  
    - 结合性：右结合  
    - 例：`x = 5`、`a += 3`（等价于 `a = a + 3`）
17. **lambda 表达式** `lambda`（最低优先级）  


**说明**：  
- 优先级高的运算符先执行，例如 `3 + 5 * 2` 先算 `5*2`，因为 `*` 优先级高于 `+`。  
- 结合性：大部分运算符左结合（从左到右计算），仅 `**`、`lambda` 等少数右结合。  
- 实际编程中，若不确定优先级，直接用括号 `()` 强制指定顺序更清晰，避免出错。











在编程中，**运算符（Operator）** 是一种特殊的符号或关键字，用于对一个或多个操作数（变量、值等）执行特定的运算或操作，并返回一个结果。  

简单说，运算符就像“操作指令”，比如数学里的“+”“-”，或者判断“等于”的“==”，它们通过规定的逻辑处理数据，输出运算结果。  


### 运算符的核心特征：  
1. **必须有操作数**：至少作用于一个数据（如取反运算符`not`作用于一个布尔值），多的可作用于两个或多个（如`+`作用于两个数）。  
2. **有明确的运算规则**：比如“+”表示加法，“>”表示比较大小，“and”表示逻辑与。  
3. **返回运算结果**：运算后一定会产生一个新的值或状态（如`3 + 5`返回`8`，`5 > 3`返回`True`）。  


### 常见分类（以Python为例）：  
- **算术运算符**：`+`（加）、`-`（减）、`*`（乘）、`/`（除）等，处理数值计算。  
- **比较运算符**：`==`（等于）、`>`（大于）、`<=`（小于等于）等，返回布尔值（`True`/`False`）。  
- **逻辑运算符**：`and`（与）、`or`（或）、`not`（非），处理逻辑判断。  
- **赋值运算符**：`=`、`+=`、`*=`等，给变量赋值或更新变量值。  
- **位运算符**：`&`（按位与）、`|`（按位或）等，直接操作二进制位。  

比如`a + b`中，`+`是运算符，`a`和`b`是操作数，运算结果是`a`与`b`的和——这就是运算符的基本作用。
