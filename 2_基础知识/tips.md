# 关于布尔值能否作为字典的键，总结如下：

1. 布尔值（`True`、`False`）可以作为字典的键，因为它们是不可变类型（符合字典键的要求）。  
2. 但需注意：在Python中，`True`本质上是整数`1`的子类（`True == 1`为`True`），`False`是整数`0`的子类（`False == 0`为`True`）。因此，若字典中同时存在布尔值键和对应的整数键（如`True`和`1`、`False`和`0`），会被视为重复键，后出现的键值对会覆盖前者。  
3. 实际使用中很少用布尔值做键，原因是不够直观，且容易与整数键冲突。更常用的键类型是字符串、整数、元组等，其中字符串因语义清晰而最常用。


# 在Python中，任何数据类型转布尔值的规则可凝练为：**“空/零为False，非空/非零为True”**。  
- 数值中，0（含0.0、0j）为False，非0为True；  
- 容器（列表、字典等）中，空容器为False，非空为True；  
- 字符串中，空串为False，非空（含空格）为True；  
- 特殊值None为False。  

本质是“是否存在有效内容/意义”的判断。
