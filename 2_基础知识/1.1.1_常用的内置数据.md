### 变量与内置数据类型  

在 Python 中，变量是存储数据的载体，其数据类型会根据所赋的值动态改变（例如`a = 5`时`a`是`int`类型，重新赋值`a = "hi"`后，`a`自动变为`str`类型）。这种动态类型特性让 Python 编程更灵活。  

#### 基础类型  
基础类型用于表示单一值，包括数值型、字符串型和布尔型，其中数值型又细分为整数、浮点数和复数：  

1. **整数（`int`）**  
   表示没有小数部分的整数，支持正负数，且大小无理论限制（可处理极大整数）。  
   ```python  
   num1 = 42       # 正整数  
   num2 = -100     # 负整数  
   num3 = 12345678901234567890  # 超大整数  
   ```  

2. **浮点数（`float`）**  
   表示带小数部分的数值，支持科学计数法（如`1.2e3`表示`1200`）。注意因二进制存储特性，可能存在精度误差。  
   ```python  
   pi = 3.14159    # 普通浮点数  
   temp = -2.5     # 负浮点数  
   large = 1.2e3   # 等价于 1200.0  
   print(0.1 + 0.2)  # 输出 0.30000000000000004（精度误差示例）  
   ```  

3. **布尔值（`bool`）**  
   仅含`True`（真）和`False`（假）两个值，用于逻辑判断，可由比较运算或逻辑运算生成。  
   ```python  
   is_ok = True  
   has_error = False  
   print(5 > 3)        # 输出 True（比较运算结果）  
   ```  

4. **字符串（`str`）**  
   由字符组成的序列，用单引号（`'`）、双引号（`"`）或三引号（`'''`/`"""`）定义，支持多行文本和丰富的操作方法（如`upper()`转大写、`split()`分割等）。  
   ```python  
   s1 = 'hello'        # 单引号定义  
   s2 = "world"        # 双引号定义  
   s3 = """多行  
   字符串"""           # 三引号支持换行  
   print(s1.upper())   # 输出 "HELLO"  
   ```  

#### Python 基本数据类型分类及核心说明
| 类型大类 | 具体类型 | 标识 | 核心特点 & 示例 |
| ---- | ---- | ---- | ---- |
| **数值型** | 整数 | `int` | 无小数部分的数值，支持正负，示例：`10`、`-25`、`0` |
|  | 浮点数 | `float` | 带小数部分的数值，示例：`3.14`、`-0.5`、`2.0` |
| **布尔型** | 布尔值 | `bool` | 只有`True`和`False`两个值，本质是`int`子类，示例：`True`（对应1）、`False`（对应0） |
| **字符串型** | 字符串 | `str` | 表示文本数据，需用引号包裹，示例：`"hello"`、`'Python'`、`'123'` |






#### 容器类型  
容器类型用于存储多个元素，根据特性可分为：  

1. **元组（`tuple`）**  
   有序但不可变的序列，元素类型可混合，常用于存储不可修改的固定数据（如坐标、配置项）。  
   ```python  
   point = (10, 20)  # 坐标数据  
   print(point[0])   # 输出 10  
   # point[0] = 15   # 报错：元组不可修改  
   ```  

2. **列表（`list`）**  
   有序且可变的序列，元素类型可混合，通过索引访问（从0开始），支持`append()`（添加元素）、`pop()`（删除元素）等操作。  
   ```python  
   fruits = ['apple', 3, True]  # 混合类型元素  
   fruits.append('banana')      # 添加元素  
   print(fruits)  # 输出 ['apple', 3, True, 'banana']  
   fruits.pop(1)  # 删除索引1的元素（3）  
   ```  


3. **集合（`set`）**  
   无序且无重复元素的集合，支持去重和数学集合运算（并集、交集等）。  
   ```python  
   nums = {1, 2, 2, 3}  # 自动去重，结果为 {1, 2, 3}  
   s1 = {1, 2, 3}  
   s2 = {3, 4, 5}  
   print(s1 & s2)  # 交集，输出 {3}  
   print(s1 | s2)  # 并集，输出 {1, 2, 3, 4, 5}  
   ```  

4. **字典（`dict`）**  
   无序的键值对集合（Python 3.7+ 后保留插入顺序），键唯一且不可变，通过键快速访问值，适合存储关联数据。  
   ```python  
   person = {'name': 'Alice', 'age': 25}  
   print(person['name'])  # 输出 "Alice"  
   person['age'] = 26     # 修改值  
   person['city'] = 'Beijing'  # 添加新键值对  
   ```  


##### Python 容器数据类型分类及核心说明
| 类型       | 定义符号/方式               | 是否有序 | 是否可变 | 元素是否可重复 | 其他核心特性                                                                 |
|------------|----------------------------|----------|----------|----------------|------------------------------------------------------------------------------|
| 元组（tuple）| `()` 或 `tuple()`（单元素需加逗号，如`(1,)`） | 是（按插入顺序） | 否       | 是             | 元素类型可混合，不可修改元素，但元素若为可变类型（如列表），其内部可改         |
| 列表（list） | `[]` 或 `list()`           | 是（按插入顺序） | 是       | 是             | 元素类型可混合，支持索引、切片，常用`append()`、`pop()`等方法修改元素         |
| 集合（set） | `{元素}` 或 `set()`（空集合必须用`set()`） | 否       | 是       | 否             | 元素需为不可变类型，支持交集（`&`）、并集（`\|`）等集合运算，无索引             |
| 字典（dict） | `{key: value}` 或 `dict()` | 是（Python 3.7+按插入顺序） | 是       | 键不可重复，值可重复 | 通过键访问值，键需为不可变类型（如字符串、数字），支持`keys()`、`values()`等方法 |



#### 特殊类型
`None` 是 Python 中一个非常特殊的存在，它属于单独的 `NoneType` 类型，而且全程序中只有一个实例（也就是说，不管在哪里用 `None`，都是同一个对象）。  

它的核心意义是“空”，但和空字符串 `""`、空列表 `[]` 这些不一样——那些是空的“容器”，而 `None` 表示“不存在值”或“没有结果”。比如：  
- 函数没有明确写 `return` 语句时，默认返回 `None`（像刚才提到的，如果一个函数只做了些操作没返回东西，调用后得到的就是 `None`）；  
- 可以用 `None` 给变量“占位”，表示这个变量暂时还没实际的值，比如 `data = None`，之后再根据情况赋值；  
- 判断一个变量是不是 `None` 时，要用 `is` 而不是 `==`，比如 `if x is None:`，这是 Python 的规范用法，因为 `None` 是单例对象，`is` 检查的是对象身份，更准确。  

简单说，`None` 就像一个“空标记”，告诉程序“这里没东西”，在处理函数返回、变量初始化时特别常用，是 Python 里不可或缺的一个特殊值～

可以按照“是否可变”和“是否有序”这两个维度，将列表（list）、元组（tuple）、集合（set）、冻结集合（frozenset）划分成四个象限，刚好对应四种组合：

| 维度       | 有序               | 无序               |
|------------|--------------------|--------------------|
| **可变**   | 列表（list）       | 集合（set）        |
| **不可变** | 元组（tuple）      | 冻结集合（frozenset） |


### 各象限特性拆解：
1. **有序 + 可变（列表 list）**  
   - 元素有固定顺序，可通过索引访问（如 `lst[0]`）。  
   - 支持增删改元素（`append()`、`pop()`、`lst[1] = 5` 等），自身结构可动态变化。  

2. **无序 + 可变（集合 set）**  
   - 元素无固定顺序，不能通过索引访问。  
   - 支持增删元素（`add()`、`remove()` 等），但元素不可重复，常用于去重或集合运算（交集、并集）。  

3. **有序 + 不可变（元组 tuple）**  
   - 元素有固定顺序，可通过索引访问（如 `tpl[2]`）。  
   - 一旦创建，不能增删改元素，结构固定，适合存储不可修改的序列（如坐标 `(x, y)`）。  

4. **无序 + 不可变（冻结集合 frozenset）**  
   - 元素无固定顺序，不能通过索引访问。  
   - 一旦创建，元素不可增删改，且元素不可重复，可作为字典的键（因不可变），适合存储需要哈希且不允许修改的集合数据。  

这样的划分刚好覆盖了 Python 中四种基础容器类型的核心差异，也解释了它们各自的适用场景——有序结构适合按序列访问，无序结构适合去重/集合运算；可变结构适合动态修改，不可变结构适合存储稳定数据或作为字典键。
