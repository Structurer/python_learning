### 常用的内置数据类型  

在 Python 中，变量是存储数据的载体，其数据类型会根据所赋的值动态改变（例如`a = 5`时`a`是`int`类型，重新赋值`a = "hi"`后，`a`自动变为`str`类型）。这种动态类型特性让 Python 编程更灵活。  

#### 基础类型  
基础类型用于表示单一值，包括数值型、字符串型和布尔型，其中数值型又细分为整数、浮点数和复数：  

1. **整数（`int`）**  
   表示没有小数部分的整数，支持正负数，且大小无理论限制（可处理极大整数）。  
   ```python  
   num1 = 42       # 正整数  
   num2 = -100     # 负整数  
   num3 = 12345678901234567890  # 超大整数  
   ```  

2. **浮点数（`float`）**  
   表示带小数部分的数值，支持科学计数法（如`1.2e3`表示`1200`）。注意因二进制存储特性，可能存在精度误差。  
   ```python  
   pi = 3.14159    # 普通浮点数  
   temp = -2.5     # 负浮点数  
   large = 1.2e3   # 等价于 1200.0  
   print(0.1 + 0.2)  # 输出 0.30000000000000004（精度误差示例）  
   ```  

3. **布尔值（`bool`）**  
   仅含`True`（真）和`False`（假）两个值，用于逻辑判断，可由比较运算或逻辑运算生成。  
   ```python  
   is_ok = True  
   has_error = False  
   print(5 > 3)        # 输出 True（比较运算结果）  
   ```  

4. **字符串（`str`）**  
   由字符组成的序列，用单引号（`'`）、双引号（`"`）或三引号（`'''`/`"""`）定义，支持多行文本和丰富的操作方法（如`upper()`转大写、`split()`分割等）。  
   ```python  
   s1 = 'hello'        # 单引号定义  
   s2 = "world"        # 双引号定义  
   s3 = """多行  
   字符串"""           # 三引号支持换行  
   print(s1.upper())   # 输出 "HELLO"  
   ```  

#### Python 基本数据类型分类及核心说明
| 类型大类 | 具体类型 | 标识 | 核心特点 & 示例 |
| ---- | ---- | ---- | ---- |
| **数值型** | 整数 | `int` | 无小数部分的数值，支持正负，示例：`10`、`-25`、`0` |
|  | 浮点数 | `float` | 带小数部分的数值，示例：`3.14`、`-0.5`、`2.0` |
| **布尔型** | 布尔值 | `bool` | 只有`True`和`False`两个值，本质是`int`子类，示例：`True`（对应1）、`False`（对应0） |
| **字符串型** | 字符串 | `str` | 表示文本数据，需用引号包裹，示例：`"hello"`、`'Python'`、`'123'` |







#### 容器类型  
容器类型用于存储多个元素，根据特性可分为：  

1. **元组（`tuple`）**  
   有序但不可变的序列，元素类型可混合，常用于存储不可修改的固定数据（如坐标、配置项）。  
   ```python  
   point = (10, 20)  # 坐标数据  
   print(point[0])   # 输出 10  
   # point[0] = 15   # 报错：元组不可修改  
   ```  

2. **列表（`list`）**  
   有序且可变的序列，元素类型可混合，通过索引访问（从0开始），支持`append()`（添加元素）、`pop()`（删除元素）等操作。  
   ```python  
   fruits = ['apple', 3, True]  # 混合类型元素  
   fruits.append('banana')      # 添加元素  
   print(fruits)  # 输出 ['apple', 3, True, 'banana']  
   fruits.pop(1)  # 删除索引1的元素（3）  
   ```  


3. **集合（`set`）**  
   无序且无重复元素的集合，支持去重和数学集合运算（并集、交集等）。  
   ```python  
   nums = {1, 2, 2, 3}  # 自动去重，结果为 {1, 2, 3}  
   s1 = {1, 2, 3}  
   s2 = {3, 4, 5}  
   print(s1 & s2)  # 交集，输出 {3}  
   print(s1 | s2)  # 并集，输出 {1, 2, 3, 4, 5}  
   ```  

4. **字典（`dict`）**  
   无序的键值对集合（Python 3.7+ 后保留插入顺序），键唯一且不可变，通过键快速访问值，适合存储关联数据。  
   ```python  
   person = {'name': 'Alice', 'age': 25}  
   print(person['name'])  # 输出 "Alice"  
   person['age'] = 26     # 修改值  
   person['city'] = 'Beijing'  # 添加新键值对  
   ```  


##### Python 容器数据类型分类及核心说明
| 类型       | 定义符号/方式               | 是否有序 | 是否可变 | 元素是否可重复 | 其他核心特性                                                                 |
|------------|----------------------------|----------|----------|----------------|------------------------------------------------------------------------------|
| 元组（tuple）| `()` 或 `tuple()`（单元素需加逗号，如`(1,)`） | 是（按插入顺序） | 否       | 是             | 元素类型可混合，不可修改元素，但元素若为可变类型（如列表），其内部可改         |
| 列表（list） | `[]` 或 `list()`           | 是（按插入顺序） | 是       | 是             | 元素类型可混合，支持索引、切片，常用`append()`、`pop()`等方法修改元素         |
| 集合（set） | `{元素}` 或 `set()`（空集合必须用`set()`） | 否       | 是       | 否             | 元素需为不可变类型，支持交集（`&`）、并集（`\|`）等集合运算，无索引             |
| 字典（dict） | `{key: value}` 或 `dict()` | 是（Python 3.7+按插入顺序） | 是       | 键不可重复，值可重复 | 通过键访问值，键需为不可变类型（如字符串、数字），支持`keys()`、`values()`等方法 |

## 次常用的内置数据类型


在 Python 内置数据类型中，以下属于日常开发中不常用（或仅在特定场景使用）的类型：

### 1. 复数（complex）
- **定义**：由实部和虚部组成的数值类型，虚部以 `j` 或 `J` 结尾，格式为 `a + bj`（`a` 为实部，`b` 为虚部，均为浮点数）。  
- **示例**：`3 + 4j`、`-1.5j`（实部为0）、`complex(2, 5)`（通过构造函数创建，等价于 `2 + 5j`）。  
- **使用场景**：仅用于科学计算、工程数学（如信号处理、量子力学）等需要处理虚数的领域。普通业务逻辑（如数据处理、Web开发）中几乎用不到。  
- **特点**：支持基本算术运算（加减乘除），可通过 `.real` 和 `.imag` 属性获取实部和虚部（如 `(3+4j).real` 返回 `3.0`）。  


### 2. 字节串（bytes）与字节数组（bytearray）
- **定义**：  
  - `bytes`：不可变的二进制序列，元素是 0-255 的整数（对应 ASCII 码或二进制数据）。  
  - `bytearray`：可变的二进制序列，功能与 `bytes` 类似，但支持动态修改字节元素。  
- **示例**：  
  - `bytes`：`b'hello'`（对应 ASCII 码的字节序列）、`bytes([0x61, 0x62])`（等价于 `b'ab'`）。  
  - `bytearray`：`bytearray(b'hello')`，可通过 `append(0x66)` 添加字节（`0x66` 对应字符 `f`）。  
- **使用场景**：用于处理底层二进制数据，如读取二进制文件（图片、视频）、网络传输中的字节流、加密算法中的原始数据等。若不涉及“字节级”操作（如仅处理文本），则无需使用。  
- **特点**：  
  - `bytes` 不可变，类似字符串的“二进制版本”，支持切片但不可修改单个字节。  
  - `bytearray` 可变，支持 `append()`、`pop()` 等修改方法，更灵活但性能略低于 `bytes`。  


### 3. 冻结集合（frozenset）
- **定义**：不可变的集合类型，元素不可重复且需为不可变类型（如 `int`、`str`、`tuple`），创建后无法增删改元素。  
- **示例**：`frozenset({1, 2, 3})`、`frozenset([1, 2, 3])`（通过可迭代对象创建）。  
- **使用场景**：仅在需要“不可变集合”的特殊场景中使用，例如：  
  - 作为字典的键（字典键要求不可变，普通 `set` 因可变无法作为键）。  
  - 作为其他集合（`set` 或 `frozenset`）的元素（集合元素必须不可变）。  
  - 确保集合内容不被意外修改（如多线程环境中的常量集合）。  
- **特点**：支持集合运算（交集 `&`、并集 `|`、差集 `-` 等），但无 `add()`、`remove()` 等修改方法，操作后会生成新的 `frozenset`。结合列表（list）、元组（tuple）、集合（set），这四种类型可按“有序/无序”和“可修改/不可修改”划分为四类：  
  - 有序且可修改：列表（list）；  
  - 有序且不可修改：元组（tuple）；  
  - 无序且可修改：集合（set）；  
  - 无序且不可修改：冻结集合（frozenset）。  
  日常去重或集合操作更常用可变的 `set`，而 `frozenset` 则用于需要“不可变无序集合”的特殊场景。  


### 4. 内存视图（memoryview）
- **定义**：用于高效访问缓冲区（如 `bytes`、`bytearray` 或其他支持缓冲区协议的对象）的“视图”，无需复制数据即可操作底层字节。  
- **示例**：  
  ```python
  data = bytearray(b'abcdef')
  mv = memoryview(data)
  print(mv[2:5])  # 输出 b'cde'（直接访问 data 的字节，不复制）
  ```  
- **使用场景**：仅在处理大型二进制数据（如超大文件、图像像素数据）时用于优化性能，避免因复制数据导致的内存开销。普通开发中几乎用不到。  
- **特点**：通过索引、切片直接操作底层缓冲区，修改视图会同步影响原始数据（如 `mv[0] = 0x67` 会修改 `data` 的第一个字节为 `0x67`，即字符 `g`）。  


### 5. NoneType（`None` 所属类型）
- **定义**：`None` 是 Python 中表示“空值”的特殊常量，其类型为 `NoneType`（通过 `type(None)` 可查看），且全程序中只有一个 `None` 实例（单例）。  
- **示例**：`x = None`、函数无返回值时默认返回 `None`（如 `def func(): pass`，调用 `func()` 返回 `None`）。  
- **使用场景**：`None` 本身常用（如初始化变量、判断函数返回值），但作为“类型”的 `NoneType` 极少单独使用，仅在类型检查（如 `isinstance(None, NoneType)`）等特殊场景中出现。  
- **特点**：`None` 与任何值（包括空字符串 `""`、空列表 `[]`）都不相等，判断时需用 `is`（如 `if x is None`）而非 `==`，这是 Python 的规范用法。  


这些类型的设计初衷是覆盖特殊场景（底层开发、科学计算、性能优化等），日常开发中无需深入掌握，只需了解其大致用途即可。重点还是放在 `int`、`str`、`list`、`dict` 等常用类型上，确保能熟练处理大部分业务逻辑。
